# HardwareMonitor

WPF-приложение для мониторинга аппаратного обеспечения компьютера с использованием технологии Windows Management Instrumentation (WMI).

## Функциональные возможности

- **Процессор**: модель, производитель, архитектура, количество ядер и потоков, базовая частота, текущая загрузка (%)
- **Оперативная память**: общий и доступный объём, процент использования, информация о каждом модуле памяти (производитель, объём, частота)
- **Диски**: 
  - Физические диски: модель, размер, тип носителя
  - Логические диски: буква, общий размер, свободное место, файловая система, процент заполнения
- **Видеокарта**: название, объём видеопамяти, версия драйвера, разрешение экрана, частота обновления
- **Сетевые адаптеры**: имя, MAC-адрес, статус подключения, скорость
- **Системная информация**: название и версия ОС, архитектура, имя компьютера, текущий пользователь
- **Процессы**: список запущенных процессов с ID, именем, статусом, загрузкой CPU и использованием памяти; возможность завершить выбранный процесс

## Автоматическое обновление

Данные обновляются автоматически каждые 3 секунды. Обновление выполняется в фоновых потоках, поэтому интерфейс остаётся отзывчивым.

## Экспорт данных

Поддерживается экспорт в три формата:
- **TXT** – полный отчёт о системе в читаемом виде
- **CSV** – таблица процессов (для анализа в Excel)
- **JSON** – все данные в структурированном формате

## Технологии

- **Платформа**: .NET Framework 4.8
- **Графический интерфейс**: WPF (Windows Presentation Foundation)
- **Архитектурный паттерн**: MVVM (Model-View-ViewModel)
- **Доступ к системной информации**: WMI (System.Management)
- **Язык запросов**: WQL (WMI Query Language)
- **Многопоточность**: async/await, Task.Run
- **Экспорт JSON**: Newtonsoft.Json

## Используемые WMI-классы

| Класс | Назначение |
|-------|------------|
| Win32_Processor | Характеристики процессора |
| Win32_PerfFormattedData_PerfOS_Processor | Текущая загрузка CPU |
| Win32_OperatingSystem | Общий и свободный объём RAM |
| Win32_PhysicalMemory | Информация о модулях памяти |
| Win32_DiskDrive | Физические диски |
| Win32_LogicalDisk | Логические разделы |
| Win32_VideoController | Видеокарта |
| Win32_NetworkAdapter | Сетевые адаптеры (PhysicalAdapter=true) |
